# Byungwook Jeon – Personal Portfolio

A personal portfolio website designed to showcase my work, background, and development approach with a focus on clarity, calm interactions, and human-friendly design.

This site reflects how I think about software: structured, intentional, and easy to understand.

## Live Site

https://bkjeon.vercel.app

---

## Overview

This portfolio presents:
- A concise personal introduction
- Selected projects with live previews and source code
- A clear breakdown of my background and approach
- A contact form for direct communication

The design is inspired by minimal, editorial-style portfolios, emphasizing readability, spacing, and subtle motion rather than visual noise.

---

## Tech Stack

**Frontend**
- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion

**Styling & Design**
- Custom CSS variables for theme consistency
- Fraunces & Epilogue fonts
- Responsive, mobile-first layout
- Soft accent color system

**Functionality**
- EmailJS for contact form handling
- Client-side form validation and feedback
- Smooth scrolling and animated menu overlay

**Deployment**
- Vercel

---

## Features

- Responsive layout optimized for desktop and mobile
- Fullscreen animated navigation menu
- Project cards with hover previews
- Direct resume download
- Contact form with success and error handling
- Consistent spacing and section alignment across pages
